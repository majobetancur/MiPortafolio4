Variables
¿Que son?
¿Como funcionan?
Ejemplos
Una variable es un espacio de memoria donde se almacena un dato o un valor durante se ejecuta un codigo, cada variable tiene un nombre único que permite usar su contenido tantas veces como se necesite, su estructura contiene un nombre (para identificarla), un valor (el dato guardado) y un tipo (texto, número, etc.) Por ejemplo: if Numero= prompt("ingrese un Numero"), al agregar prompt estamos pidiendo que se ingrese un dato a la variable
Condicionales
¿Que son?
¿Como funciona?
Ejemplos
Es una estructura de control de flujo que permite ejecutar diferentes secciones de codigo en funcion de una condicion que se evalua como verdadera o falsa. Ademas, son valores que permanecen por mucho tiempo y cambian cada cierto tiempo. Se le tiene que dar el valor a un dato como por ejemplo, pi=3,14 ya que pi no cambiara y su valor permanece asi

Segundos temas

Switch case
¿Que es?
¿Como funciona?
Ejemplos
Es una estructura que permite ejecutar bloques de codigos mediante una variable, es un reemplazo al if en el caso de que hayan muchas variables. Por ejemplo para evitar usar if, else if, else, etc mejor se usa Switch case
Ciclo while
¿Que es?
¿Como funciona?
Ejemplo
Un ciclo while es una estructura que sirve para que cierta parte de un codigo se repita hasta que la condicion no se cumpla, por ejemplo si se quiere jugar piedra, papel o tijera ese codigo se podria repetir las veces que se introduzcan usando el ciclo while
